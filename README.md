# Implemetacion-AFD

### Ejecutar el programa
Usar el siguiente comando pasando el nombre del archivo como parametro:

```bash
python AFD.py entrada.txt
```

### Salida 
El programa procesara cada linea y mostrara en la consola el resultado del analisis:

ACEPTA: Si la cadena comienza con 0.

NO ACEPTA: Si la cadena comienza con 1, esta vacia o contiene caracteres no validos.
